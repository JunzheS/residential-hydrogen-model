# Residential Hydrogen Energy Model

Techno-economic model for residential hydrogen energy systems.

The model evaluates the feasibility of hydrogen production from rooftop PV systems for residential heating.

## Features

- PV system sizing
- Hydrogen production estimation
- Hydrogen storage requirement
- Hydrogen storage volume estimation
- Levelized Cost of Hydrogen (LCOH)
- Heating cost comparison
- Sensitivity analysis
- Hydrogen competitiveness analysis
- Interactive web tool

## System configuration

PV → Electrolyser → Hydrogen storage → Hydrogen boiler → Heat supply

## Outputs

- Hydrogen production
- PV installation area
- Required hydrogen storage
- Hydrogen storage volume
- LCOH
- Heating cost comparison
- Sensitivity plots
